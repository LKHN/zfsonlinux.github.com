NAME

zvol_wait - Wait for ZFS volume links in /dev to be created.  

SYNOPSIS

 

DESCRIPTION

When a ZFS pool is imported, ZFS will register each ZFS volume (zvol) as a disk device with the system. As the disks are registered, udev7 will asynchronously create symlinks under /dev/zvol using the zvol's name. will wait for all those symlinks to be created before returning.
